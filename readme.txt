Follow these steps to get started with Lemoon
=============================================
* Configure the connection string in Web.config to point to an empty database
* Compile and run the application (Ctrl+F5)

Hints and more information
=============================================
* Consider moving (or excluding) App_Data\Lib from your project (the files in that folder do not need to be deployed)
* Visit http://www.lemoon.com/developers for more information about developing with Lemoon
